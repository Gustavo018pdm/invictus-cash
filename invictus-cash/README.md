# Invictus Cash

Um website moderno e responsivo para a plataforma financeira **Invictus Cash** - Soluções Financeiras do Futuro.

## 🚀 Funcionalidades

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Animações Suaves**: Efeitos de entrada e transições elegantes
- **Navegação Intuitiva**: Menu fixo com links smooth scroll
- **Hero Section**: Apresentação impactante com mockup de app
- **Features**: 6 funcionalidades principais destacadas
- **Soluções**: Cards de soluções financeiras futuras
- **Recursos**: Seção educacional
- **CTA**: Chamadas para ação estratégicas
- **Footer**: Links de suporte e redes sociais

## 🔧 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/Gustavo018pdm/invictus-cash.git
   cd invictus-cash
   ```

2. **Abra no navegador**:
   - Simplesmente abra o arquivo `index.html` em seu navegador, ou
   - Use um servidor local (exemplo com Python):
     ```bash
     python -m http.server 8000
     ```
   - Acesse `http://localhost:8000`

3. **Personalize**:
   - Atualize imagens na pasta `images/`
   - Modifique cores em `css/styles.css` (variáveis CSS)
   - Adicione sua lógica no `js/script.js`
